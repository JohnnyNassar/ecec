Display vs. source The "displays at" number is what you'll see on screen. The "upload ≥" number is the minimum source size for a sharp render on retina displays — roughly 2× the display size.
Image sourcing JPG under ~800 KB. Every image is cover-cropped to the block's aspect ratio — centre the subject when preparing sources. Uploading bigger is fine; WordPress scales down automatically.
Optional blocks Every block is optional. A short project may use 2–3 blocks; a flagship project 8+. Order is drag-and-drop in the admin Content Blocks editor.
