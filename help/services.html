01

Page overview

The Our Services page is built in Elementor on the our-services page. It promotes the Emaurri demo's service-list pattern and was tuned to match ECEC's brand.

To edit anything: WP Admin → Pages → Our Services → Edit with Elementor.

✓
Recent cleanup (2026-05-17):

The faint "cozy" background watermark inherited from the Emaurri demo theme has been removed. If you ever see a giant grey word appear behind a section after a theme update, open that section in Elementor → Style tab → Background and clear the "qodef_row_background_text" field.

02

Service category blocks

Each major ECEC service (MEP, ICT, Sustainability, etc.) has its own block on the page. Each block contains:

  • An icon or image
  • A heading (the service name)
  • A short paragraph describing the service

To edit a service description

  1. Open the page in Elementor.
  2. Click on the heading or text widget for the service.
  3. Edit text in the left panel or inline.
  4. Click Update.

To swap a service icon

Click the icon widget. The left panel shows the icon picker (or image widget — depends on which type was used). Swap and save.

03

Strategic Advantages grid

The grid section showing ECEC's competitive edge: precision, sustainability, technology, etc. Each card is an Elementor inner-section column with an icon + heading + body.

Edit the same way as service blocks: click the widget, edit in left panel, save.

i
To add a new advantage card:

Right-click an existing card in Elementor, choose Duplicate. The copy appears next to the original. Edit its content. The grid auto-wraps to a new row when there are too many for one row.

04

Sectors served bar

The horizontal strip listing the sectors ECEC has worked in (Hospitality, Healthcare, Aviation, etc.). Currently styled with rounded "pill" buttons.

Each pill is an Elementor button widget inside a flex inner-section. Edit text by clicking the button, change the label in the left panel.

05

Typography & alignment

✓
Body text is justified.

The Our Services page uses justified text alignment for narratives, scoped via CSS to body.page-id-328. If you find a paragraph reads awkwardly, the answer is usually to shorten it rather than disable justification — consistency with the page is more important than one paragraph's micro-rivers.

Section headings (h2) use the brand bold uppercase style. Don't override font-weight or transform on individual widgets — it'll create inconsistency with About Us and other branded pages.
