Logos & settings.

Site-wide elements: the header logo, favicon, footer, navigation menu, SEO basics. Quick fixes that affect every page at once.

01

Header logo (the ECEC mark)

The logo at the top-left of every page. Currently shows the Emaurri demo's wordmark — needs to be replaced with the official ECEC logo.

Open the customizerWP Admin → Appearance → Customize.
Click "Site Identity"Or sometimes labelled "General" → "Logo".
Click "Change logo"WP media modal opens.
Upload the official ECEC logoUse a PNG with transparency. Recommended height: 80–120px. Width auto-scales.
Click PublishChanges appear on every page after a hard refresh.
✓
Recommended logo formats:

Header (light bg): black or dark-gray PNG with transparency, 320 × 80px, < 50KB. Footer (dark bg): white PNG (the theme inverts automatically).

02

Favicon

The little icon in the browser tab. Set in the same Customizer panel as the header logo.

  1. WP Admin → Appearance → Customize → Site Identity.
  2. Scroll to "Site Icon".
  3. Upload a square PNG, 512 × 512.
  4. Click Publish.
03

Main navigation menu

The horizontal menu at the top of every page (Home, About Us, Why ECEC, Projects, etc.).

Open the menu editorWP Admin → Appearance → Menus.
Pick the menuThe site has one menu (term_id 2) used in all 4 nav locations (main, fullscreen, vertical, mobile).
Drag to reorderItems at the top of the list appear left in the menu.
Add or remove itemsUse the left-side panel to add Pages, Custom Links or Categories.
Click Save MenuLive immediately on every page.
!
Beware of broken links:

If you delete a Page that's still in the menu, the menu item becomes a dead link. Always remove menu items first, then delete the page (or repoint the menu item).

04

Footer content

The footer is rendered by 3 column widgets (top area) + 1 widget (bottom copyright bar). Edit at:

WP Admin → Appearance → Widgets

You'll see widget areas labelled Footer Top Column 1/2/3 and Footer Bottom. Each contains a Custom HTML widget. Edit the HTML to update copy, links, or contact details.

i
Footer styling is locked:

The footer uses dark Raisin Black background with brand colors. Don't add inline color styles to footer content — they may clash with the theme. The CSS already handles all text in the footer area.

05

SEO & meta titles

Yoast SEO is installed. Each page and project has its own SEO panel below the editor.

To update a page's title or description

  1. Open the page in WP Admin.
  2. Scroll below the main editor to find the Yoast SEO panel.
  3. Edit SEO title and Meta description. Yoast shows a live preview of how it'll appear in Google.
  4. Click Update.

Sitemap

Yoast auto-generates a sitemap at /sitemap_index.xml. Submit this to Google Search Console (still pending verification at time of writing).

06

Clearing caches

If a change isn't showing on the live site, the cause is almost always a stale cache. Clear in this order:

  1. Browser cache: hard refresh with Ctrl + F5.
  2. Elementor cache: WP Admin → Elementor → Tools → Regenerate CSS & Data.
  3. If you have a caching plugin (WP Rocket, W3 Total Cache): click its "Purge All" button.

If the change still doesn't show after all three: it might be a stale CDN. Most ECEC pages don't go through a CDN, but if there's one, purge it. Otherwise try Incognito mode — that bypasses every form of browser cache.
