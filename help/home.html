The home page.

Five blocks make up the home page. Edit any of them without rebuilding from scratch — from the top blog slider to the featured projects strip at the bottom.

Layout overview

The home page is built in WP Admin → Pages → Home (post 119). It uses Elementor with five top-level sections, top to bottom.

Block 1 — Blog sliderLatest 4 blog posts in a slider with pagination dots. Auto-updates when you publish blog posts.
Block 2 — "We Design the Future"2-column intro: heading on the left, narrative text on the right.
Block 3 — Main RevSliderFull-width hero slider with project images. Edited via WP Admin → Slider Revolution.
Block 4 — Clients marqueeAuto-scrolling logo strip with "Our Clients" heading. Logos managed centrally.
Block 5 — Featured Projects3-column grid pulled from projects flagged as Featured.
i
To edit any block:

Open WP Admin → Pages → Home, click Edit with Elementor. The 5 sections appear top to bottom. Click any section to expand and edit its widget contents.

02

Block 1 — Blog slider

This is the wide image slider at the very top of the home page. It pulls the 4 latest blog posts and turns them into slides.

To change which posts appear

Just publish a blog post (WP Admin → Posts → Add New) — it'll automatically replace the oldest of the 4. No home-page edit required.

To change the slider behaviour (speed, dots)

Open the home page in Elementor, click the Blog list widget in Block 1. The right panel shows fields for slider speed, pagination, autoplay, etc.

03

Block 2 — "We Design the Future"

The 2-column intro section. The HTML is hardcoded inside an Elementor HTML widget — meaning the heading and narrative are stored as HTML, not as editable Elementor fields.

Open the home page in ElementorFrom WP Admin → Pages → Home → Edit with Elementor.
Click on Block 2's HTML widgetIt's the second section. The widget contents appear in the left panel.
Edit the HTML directlyYou'll see <h2 class="ecec-home-intro__heading">We Design the Future</h2> — change the text inside the tags. Don't change the class names.
Click UpdateThe new text appears on the live site immediately.
!
Don't change the HTML structure.

Keep __heading and __body classes intact — the CSS styling depends on them. Only change the text between the tags.

04

Block 3 — Main slider (RevSlider)

The big hero slider with project images. Editing it lives in WP Admin → Slider Revolution, not the home page itself.

Open Slider RevolutionFrom the WP admin sidebar.
Click "main-home"That's the slider used on the home page.
Edit individual slidesEach slide has a background image, optional text overlays, and timing controls. Drag to reorder.
Click SaveThe new slider appears on the home page after a hard refresh.
i
Image specs:

Use 1920 × 1080 JPGs for slider backgrounds — smaller files cause noticeable upscaling. Keep the file under 500 KB per slide for fast loads.

05

Block 4 — Clients marquee

The auto-scrolling logo strip with "Our Clients" heading. The list of logos is shared between the home page and the About Us page — edit once, both update.

To add or remove a client logo

Logos live in the WP media library, and a saved option (ecec_client_logo_ids) tracks which ones to show.

  1. Upload the new logo to WP Admin → Media → Add New. Use a transparent PNG, height around 80–120px.
  2. Note the new attachment's ID (visible in the URL of the media item).
  3. Run _deploy/update_client_logos.php with the updated list of IDs — or use the management interface if available.
✓
Heading override:

The "Our Clients" title is set in the shortcode — default is Our Clients. To change it on a specific page, override via [ecec_clients_marquee title="Our Partners"]. Pass title="" to hide it.

06

Block 5 — Featured Projects

The 3-card grid at the bottom of the home page. Each card is a project flagged as Featured.

To change which projects appear

  1. Open WP Admin → Portfolio.
  2. Click any project to open it.
  3. In the right sidebar under Project options, toggle the Featured project checkbox.
  4. Set Menu Order — lower numbers appear first.
  5. Click Update.
!
Limit:

Block 5 shows the first 3 featured projects (sorted by Menu Order ascending). The Projects page also has a Featured strip but it shows up to 6.

For step-by-step on toggling Featured and using Menu Order, see the Projects guide · Mark as Featured.

FAQ

I edited a block but the home page still shows the old version.

Two-step fix: (1) hard refresh in your browser with Ctrl + F5; (2) clear Elementor's CSS cache from Elementor → Tools → Regenerate CSS & Data.

The clients marquee logos look different sizes.

Make sure each logo is exported with consistent height (e.g. 100px tall) and transparent background. Different intrinsic heights cause uneven rendering even though our CSS forces a row height.

Where do I edit the page background or section colors?

Open the home page in Elementor → click any section's handle → in the left panel switch to Style tab → background color/image. Save.
