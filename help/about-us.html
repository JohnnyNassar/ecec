02

Section 1 — Chairs hero

The wide image at the top of the About Us page. As of 2026-05-17 the page title text is hidden on this page — only the chairs photo and the breadcrumbs are shown over the band. The hero is configured via the page's Qode title settings, not Elementor.

To change the WP page title (used in tabs / search / nav)

Open WP Admin → Pages → About Us → Edit (the WP editor, not Elementor). The title at the top of the screen is the WordPress page title; it won't show in the hero band but it's still used for the browser tab and SEO. Change it and click Update.

To swap the chairs image

  1. Upload your new image to WP Admin → Media. Use a wide landscape JPG, ideally 2000 × 800.
  2. Note the new attachment URL (right-click image, copy URL).
  3. Replace the file at /wp-content/uploads/about-hero/about-chairs.jpg via FTP / file manager — OR update the Qode "page title image" field for this page in Edit page → Sidebar → Title area.

Section 5 — Key Highlights cards

The 4-card grid at the bottom of the page. Each card has a PNG number (No1, No2, No3, No4), a heading and a body paragraph.

To edit a card's text

In Elementor, click any card's heading or body and edit inline. The PNG number is an image widget — click it to swap.

To swap the number PNGs

The numbers live at /wp-content/uploads/about-numbers/No1.png through No4.png. To restyle, replace those files (same dimensions) and refresh.

i
Card body alignment:

Body text is centered (changed 2026-05-17) under each numbered card. Previously it was justified; the columns were too narrow for justified text to look clean. If you ever switch back, also enable hyphenation or paragraphs get unsightly gaps.

FAQ

The page renders but the chairs image is missing.

The image must exist at /wp-content/uploads/about-hero/about-chairs.jpg. Check via FTP / file manager. If it's there but still missing, hard refresh and clear Elementor cache.

I want to add a 5th highlight card.

The CSS uses a 4-column grid. Adding a 5th card would put one alone on a second row. Either replace one of the existing four, or have a developer change the grid to 5 columns and add a No5.png.

Old "About Us" content is showing.

Make sure your menu's About Us link points to page 382, not page 120 (the archived version). Check via WP Admin → Appearance → Menus → About Us → URL.
