Three rules that
always apply.

  • Always preview. Click Preview in the top-right of any edit screen before clicking Update. Catches mistakes before they go live.
  • Drafts are free. If you're unsure, set the post status to Draft while editing. It won't show on the public site until you publish.
  • Hard-refresh after CSS. If something looks off after a deploy, press Ctrl + F5 — you're seeing cached files.
  • Image specs matter. Use the dimensions noted in each guide. Wrong sizes cause layout shifts on the live site.
