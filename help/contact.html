Office addresses & phone numbers

The Contact Us page (page 124) has city blocks for each office. Each block has a city heading, address, phone, email.

Open the page in ElementorWP Admin → Pages → Contact Us → Edit with Elementor.
Click any city's heading or text widgetEdit text in the left panel or inline.
Update phone / email linksIf editing a clickable phone number (tel:+971...) or email (mailto:), edit the link target in the widget's link settings.
Click UpdateLive immediately.
i
Footer too:

The site footer also lists offices — that's a separate widget under Appearance → Widgets (custom_html widget #12). Update both for consistency.

02

Contact form recipients

Form submissions are emailed to the configured recipients. ECEC currently routes both forms (the main contact form #132 and the older form #6) to info@ecec.co.

Open Contact Form 7WP Admin sidebar → Contact.
Click the form to editPick "Contact form 1" (132) or the legacy form.
Switch to the Mail tabTop of the form editor.
Update the To: fieldChange info@ecec.co to the new recipient. To send to multiple addresses, comma-separate.
Click SaveTest by submitting the form on the live site.
!
Always test:

After changing the recipient, submit a test message via the live form. Check the new recipient's inbox (and spam folder). If nothing arrives within a minute, check WP's mail config — the server may need an SMTP plugin like WP Mail SMTP.

03

Map embed

The map at the bottom of the Contact page is a Google Maps iframe. To replace it with a different location:

  1. Open Google Maps, search for the new address.
  2. Click Share → Embed a map.
  3. Copy the iframe HTML.
  4. In Elementor, click the map widget on the Contact page.
  5. Paste the iframe code into the HTML widget content.
  6. Click Update.
04

FAQ

Form submissions aren't arriving.

Check spam first. If still missing, the WP server may need an SMTP plugin to actually deliver mail. Install WP Mail SMTP and configure with your provider's credentials (Gmail, Mailgun, etc.).

The "Send" button doesn't do anything.

Browser console (F12 → Console) usually shows the error. Most common cause: a JS plugin conflict. Try deactivating recent plugins one at a time.

I want to add a 4th office.

In Elementor, right-click an existing office block → Duplicate. Edit the city/address/contact details on the copy. Reposition with drag.
