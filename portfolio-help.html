Part A — Home Page Slider (RevSlider)

The home page features a full-width image slider powered by Slider Revolution. The slider is named "Main Home" (alias: main-home).

1 Change a Slide Image
  • Go to Slider Revolution in the left admin sidebar
  • Click on the "Main Home" slider to open it
  • At the bottom of the editor, you'll see thumbnails of all slides — click the slide you want to edit
  • In the slide editor, look for "Slide Background" panel (usually on the right or top)
  • Click the current background image
  • Click "Media Library" to select/upload a new image
  • Recommended size: 1920 x 900 px or larger (full-width landscape)
  • Click Save at the top of the editor
Tip: Use high-quality project photos. The slider stretches images to fill the full browser width, so low-resolution images will look blurry.
2 Edit Slide Text
  • Open the slide in the editor (see Step 1)
  • You'll see the slide canvas with text layers overlaid on the image
  • Double-click on any text layer to edit it inline
  • You can change the text, font size, and color in the Layer Options panel on the right
  • To move text, drag the layer to a new position on the canvas
  • Click Save when done
Layer types: Each piece of text (title, subtitle, button) is a separate "layer". Click on each one individually to edit it.
3 Add or Remove Slides

To add a new slide:

  • In the slider editor, click the "+" button next to the slide thumbnails at the bottom
  • Choose "Blank Slide" or "Duplicate" an existing slide (recommended — keeps the same text layout)
  • Set the background image and edit the text layers
  • Click Save

To remove a slide:

  • Right-click on the slide thumbnail at the bottom
  • Select "Delete"
  • Click Save

To reorder slides:

  • Drag and drop the slide thumbnails at the bottom to reorder
  • Click Save
Important: Always click Save after making changes. Changes are not auto-saved. If you navigate away without saving, your edits will be lost.

Part B — Portfolio Projects

1 Create the Portfolio Item

From the WordPress admin dashboard:

  • Go to Portfolio → Add New in the left sidebar
  • Enter the project name as the title (e.g., "Al Riyadh North Mall")
Tip: The title will appear on the Projects page grid and the single project page header.
2 Add the Featured Image

The featured image is the main thumbnail shown on the Projects listing page.

  • In the right sidebar, find Featured Image
  • Click Set featured image
  • Upload or select an image from the Media Library
  • Recommended size: 1200 x 800 px minimum (landscape orientation)
Required: Every portfolio item must have a featured image. Without it, the project will appear as a blank card on the listing page.
3 Write the Project Description

Use the main WordPress editor (content area) to write the project description.

  • Write a detailed description of the project scope and deliverables
  • This text appears on the single project page below the media section
  • Use paragraphs for readability; avoid excessive formatting
  • The content is displayed with justified text alignment
4 Set the Portfolio Category (Sector)

Categories determine which sector filter the project appears under on the Projects page.

  • In the right sidebar, find Portfolio Categories
  • Check one or more categories that apply

Available sectors:

Commercial & Retail Data Centers Educational Governmental Sector Healthcare Hospitality Industrial & Distribution Master Planning Residential Sustainability
Tip: You can assign multiple categories. The project will appear under each selected filter on the Projects page. The category also displays as "Sector:" on the single project page.
5 Add Project Info Fields (Sidebar)

Scroll down to the Portfolio Settings meta box. Under the Info section, you'll find a repeater field for project details. These appear in the sidebar on the single project page.

How to add an info item:

  • Click "Add New Item" to add a row
  • Fill in the Label (e.g., "Client") and Value (e.g., "Modon Properties")
  • Leave Link blank unless the value should be clickable
  • Repeat for each field

Standard info fields used on ECEC projects:

Label Example Value Status
Client Modon Properties Required
Contractor Amana Group Required
Engineering MEP and ICT Required
Location Riyadh, Saudi Arabia Required
Area 125,000 sqm Optional
Order matters: The info items display in the same order you add them. Keep the order consistent: Client, Contractor, Engineering, Location, Area.
Two different "Location" fields, for two different purposes:
  • The "Location" info row above is free text that shows on the single-project sidebar (e.g. "Riyadh, Saudi Arabia", "Yas Island, Abu Dhabi"). Use it to describe the project's actual address or area.
  • The "Location" taxonomy (covered in Step 10 below) is a separate country-level tag used only by the By Location dropdown on the Projects page filter bar. It doesn't display on the single project page.
Fill in both — they're independent.
6 Add Project Media (Gallery Images)

In the Portfolio Settings meta box, find the Media section. This controls the images shown on the single project page above the description.

To add a single image:

  • Click "Add New Item"
  • Set Media Type to "Image"
  • Click the image upload button and select/upload your image
  • Recommended size: 1600 x 1000 px or larger

To add a gallery (multiple images):

  • Click "Add New Item"
  • Set Media Type to "Gallery"
  • Select multiple images from the Media Library

Other media types available:

  • Video - Paste a YouTube or Vimeo URL
  • Audio - Paste an audio file URL
Tip: You can add multiple media items (mix of images, galleries, and videos). They display in the order you add them.
7 Set the Completion Date

The publish date of the post is used as the completion date. It displays as "Completion Date:" with the year only on the single project page.

  • In the right sidebar under Publish, click the date
  • Set the year to the project completion year
  • The month/day don't matter visually (only the year shows), but they affect the sort order on the listing page
8 Publish and Verify
  • Click Publish (or Update if editing)
  • View the project by clicking View Portfolio Item
  • Check the Projects page to verify it appears in the grid
  • Click the relevant sector filter to confirm it shows under the correct category

Checklist before publishing:

  • Featured image is set (landscape, high resolution)
  • Project title is correct
  • Description is written in the content editor
  • Portfolio Category (sector) is selected
  • Info fields are filled: Client, Contractor, Engineering, Location
  • At least one media item (image/gallery) is added
  • Completion year is set via the publish date
9 Mark a Project as Featured

Featured projects appear in the Featured Projects section at the top of the Projects page (above the full grid) and display a ★ FEATURED badge on the card wherever they appear on the site.

To feature a project:

  • Open the project for editing (Portfolio → All Portfolio Items, click the project title)
  • On the right sidebar, find the "Featured project" box
  • Tick the checkbox: "Show in the Featured Projects section on the Projects page"
  • Click Update

To unfeature a project:

  • Open the project and uncheck the "Featured project" box
  • Click Update — the project vanishes from the Featured section and its badge disappears

To change the order of featured projects:

  • On the project edit screen, find Page Attributes in the right sidebar
  • Set the Order number — lower numbers appear first (e.g. 1 is the very first featured card)
  • Click Update
Tip: Leave gaps in the Order numbers (e.g. 10, 20, 30) so you can slot new featured projects between existing ones later without renumbering everything.
See all featured at a glance: Go to Portfolio → All Portfolio Items. The list has a Featured column showing a yellow ★ next to each featured project.
10 Set the Project Location (Country)

The Location taxonomy controls the By Location dropdown on the Projects page filter bar. This is a country-level tag and is separate from the free-text "Location" info row (see Step 5).

To assign a location:

  • On the project edit screen, find the Locations box in the right sidebar (below Portfolio Categories)
  • Tick one location:
Saudi Arabia UAE Jordan Other
  • Click Update

When to use "Other":

Pick Other for any project outside Saudi Arabia, UAE, or Jordan (e.g. an Egypt or Iraq project). It keeps the dropdown focused on the firm's three primary markets while still tagging the project.

To add a new country term (rare):

  • Go to Portfolio → Locations
  • Fill in Name (e.g. Qatar) and Slug (e.g. qatar)
  • Click Add New Location
  • The new option appears in the By Location dropdown on the Projects page automatically, listed after the four built-in options
Projects with no location: If a project has no location tag, it will still appear in the grid, but selecting any By Location filter will hide it. Always assign a location when you publish a new project.

Part C — Team Members (People Page)

The People page shows a grid of team members — photo, name, role, and office locations. Each member is a separate entry in the Team Members section of the admin sidebar. You can add, edit, reorder, or remove members without touching the page itself.

12 Add a New Team Member
  • In the admin sidebar, click Team Members → Add New Member
  • Enter the person's full name in the Title field (e.g. Jane Smith)
  • In the Team Member Details box, fill in:
    • Role / Title — e.g. Senior MEP Engineer or Head of ICT
    • Office Locations — e.g. Dubai | Riyadh (separate multiple offices with the | character)
  • On the right sidebar, click Set featured image and upload a photo
    • Recommended: square photo at least 500 × 500 px
    • Landscape or portrait photos will be cropped to square automatically
  • Also on the right sidebar, find Page Attributes → Order and enter a number (lower numbers appear first in the grid — e.g. 1 for the founder, 2 for the next senior member, etc.)
  • Click Publish
  • Visit the People page on the website — the new member appears in the grid immediately
Tip: You don't need to touch Elementor or any page builder. The People page automatically pulls in all published team members.
13 Edit or Remove a Member

To edit a member:

  • Go to Team Members → All Team Members
  • Hover over the member's name and click Edit
  • Change the name, photo, role, location, or order
  • Click Update

To remove a member:

  • Go to Team Members → All Team Members
  • Hover over the member's name and click Trash
  • The member disappears from the People page immediately
Note: You can also unpublish a member (set to Draft) to temporarily hide them without deleting. Draft members are not shown on the People page but stay in the admin for later.
14 Reorder the Team Grid
  • Go to Team Members → All Team Members
  • Open each member in turn and set their Order number under Page Attributes (right sidebar)
  • Lower numbers appear first (1 = first, 2 = second, etc.)
  • Click Update
Tip: Leave gaps in the numbers (e.g. 10, 20, 30) so you can insert new members between existing ones without renumbering everyone.

Part D — Projects Page Layout

The Projects page has four stacked sections, top to bottom:

  1. Hero — full-width image with headline and subtitle
  2. Tabs strip — FEATURED / RECENT PROJECTS links (click-scroll to each section)
  3. Featured Projects — 2-column grid of projects marked as featured
  4. Our Recent Projects — search/filter bar + 3-column grid of all projects
17 Edit the Hero Image, Headline, or Subtitle

The hero section at the top of the Projects page is a standard Elementor block. You can change the image, text, and styling without touching code.

To edit the hero:

  • Go to Pages → All Pages and click Projects
  • Click Edit with Elementor at the top of the page
  • The Elementor editor opens. The first section at the top (dark image with the headline) is the hero.

To change the background image:

  • Click once on the hero section (not on the text — on the image area)
  • On the left panel, switch to the Style tab
  • Under Background, click the current image thumbnail
  • Select a new image from the Media Library (or upload one)
  • Click Update at the bottom-left of the Elementor editor

To change the headline or subtitle text:

  • Click on the headline or subtitle text directly
  • Edit the text in the left panel's Content → Title field (or type directly in the canvas)
  • Click Update
Tip: Use a high-resolution landscape image (at least 1920 x 1080 px). The hero stretches to full browser width with a dark overlay, so dimly-lit or detailed-dark photos work best.
18 How the Featured & Recent Sections Work

Featured Projects section (top of page)

  • Displays only projects marked as Featured (see Step 9)
  • Shows them in a 2-column grid with large images, title, country, and "View Project →" link
  • Order is controlled by each project's Page Attributes → Order field (lower number = earlier)
  • If no projects are marked as featured, this section shows a helper message

Recent Projects section (bottom of page)

  • Displays every published portfolio item (54 projects currently)
  • Shows them in a 3-column grid with the search/filter bar above and the sector sidebar on the left
  • Order is by publish date (newest first)
  • Featured projects get a ★ FEATURED badge on their card here, but otherwise appear in the normal order
To remove a project from both sections: change its status to Draft (instead of deleting it). Draft projects disappear from the site but stay in the admin for later.
19 How the Filter Bar Works

The bar above the Recent Projects grid has four controls:

Control What it filters by How it matches
Search here Project title Case-insensitive substring (type "amaa" to find both AMAALA resorts)
By Type Portfolio Category (Sector) Exact match on the category you pick
By Location Portfolio Location (Country) Exact match on Saudi Arabia / UAE / Jordan / Other
SEARCH button Re-applies all filters Not required — filters apply live as you type / change a dropdown
Filters stack. If you select Hospitality and UAE, you see only UAE hospitality projects. If you also type "royal" in the search field, you narrow further to just Royal Atlantis. Clear a filter by setting the dropdown back to its default (By Type / By Location) or emptying the search text.
The left sidebar filter (Commercial & Retail, Data Centers, etc.) also stacks with the bar. Clicking it narrows the grid the same way.
20 Rename the FEATURED / RECENT PROJECTS Tabs

The two tab labels below the hero ("FEATURED" and "RECENT PROJECTS") are editable through the Elementor editor.

  • Go to Pages → Projects → Edit with Elementor
  • Scroll down past the hero — find the narrow strip with the two tab links
  • Click the block to select it, then on the left panel find the HTML widget
  • The HTML panel shows: 
    <nav class="ecec-projects-tabs">
  <a href="#featured-projects">Featured</a>
  <a href="#recent-projects">Recent Projects</a>
</nav>
  • Change the text between the <a> tags (keep everything else unchanged)
  • Click Update
Don't change the href values (#featured-projects and #recent-projects) — those are the anchor links that make the tabs scroll to the right sections. Changing them breaks the smooth-scroll behavior.

Part E — Services Page (Elementor)

The Services page at /our-services/ is built in Elementor, based on the Emaurri theme's demo layout with ECEC's own narrative. Every text box, number, image, and accordion item can be edited inline — no developer required.

Previous approved layout (numbered-list style) is archived at /our-services-legacy/. It's still reachable by direct URL but not linked from the main navigation. Remove it when you no longer need the backup.

21Open the Services Page in Elementor

  1. From the admin sidebar, go to Pages.
  2. Find the row titled Our Services (Preview) and click it.
  3. At the top, click the blue Edit with Elementor button.
  4. The page opens in the Elementor editor. Widgets are shown on the left; the live canvas is on the right. Click any widget on the canvas to edit its settings on the left.
  5. Always click the green UPDATE button at the bottom-left when you're done.
Tip: to see the page as visitors see it (without editor chrome), click the eye icon next to UPDATE to preview.

22Edit the Hero Title & Intro Text

The first section is the hero: a large decorative image on the left, a headline plus intro paragraph plus an explore button on the right.

  1. Click the headline text in the canvas. The left panel now shows a “Section Title” widget.
  2. Edit the Title field — that's the big line (“At ECEC, we recognize…”).
  3. Edit the Text field right below — that's the paragraph under the title.
  4. Changes show live on the canvas as you type.
  5. Click UPDATE.

23Change the Three Decorative Images

The page has three big photos: one in the hero, one beside the Progress bars, and one beside the services accordion. They're all showing the Desert Rock Resort image by default.

  1. Click the image you want to replace on the canvas.
  2. In the left panel, click the thumbnail preview under the “Image” field.
  3. The WordPress Media Library opens. Upload a new image or pick an existing one.
  4. Click Insert Media.
  5. Repeat for the other two images. Click UPDATE.
Suggested images: architectural / interior shots that feel premium. Landscape orientation works best; minimum 1600×1000 px for sharpness on large screens.

24Update the Progress Bars & Counter Numbers

The page displays three progress bars (80 projects, 60 clients, 10 sectors) and four counters (25 years, 60 partners, 50 clients, 12 active projects). All are placeholders — please replace with the real figures.

To edit a progress bar:

  1. Click the progress bar on the canvas.
  2. Change the Number field (e.g. 80).
  3. Change the Title field (e.g. projects).

To edit a counter:

  1. Click the counter number on the canvas.
  2. Change the End Digit (that's the final number the counter animates to).
  3. Change the Text field (the label underneath).

Click UPDATE when done.

25Edit, Add, Remove or Reorder Services (Accordion)

The accordion at the bottom holds all 7 services: Structural, MEP, Acoustic, BIM, ICT, Security, Supplementary. Each row has a title (ALL CAPS) and a body description.

To edit an existing service:

  1. Click any accordion row on the canvas.
  2. In the left panel, under the Accordion Items repeater, click the item you want to edit.
  3. Change the Title and the Content (rich-text).

To add a new service:

  1. Click the accordion on the canvas.
  2. At the bottom of the Accordion Items list, click + ADD ITEM.
  3. Fill in the Title and Content.

To remove a service:

  1. Click the accordion on the canvas.
  2. In the Accordion Items list, open the item and click the small × in its header.

To reorder services:

  1. Click the accordion on the canvas.
  2. In the Accordion Items list, drag items up or down using the handle at the left of each row.

Click UPDATE when done.

26Change the "Explore" Button Link

The button at the end of the hero currently links to /projects/. To retarget it:

  1. Click the explore button on the canvas.
  2. In the left panel, scroll to the Link field.
  3. Type or paste a new URL. Full URLs (https://…) and site-relative ones (/about-us/) both work.
  4. If you want the button to scroll to a section on the same page, enter #section-id (ask a developer for the section ID first).

Click UPDATE.
